## ① สรุปเนื้อหาที่เรียน — ฉันได้เรียนรู้อะไรในสัปดาห์นี้?
การใช้ terminal ทั้งของ window(CMD) และ vscode ในการรัน command และ script เบื้องต้น

## ② สรุปการฝึกปฏิบัติ — ฉันได้ทดลองและฝึกเขียนโปรแกรมอะไรบ้าง?
สัปดาห์นี้เป็นไม่ได้มีอะไรมาก แต่จะเป็นการ setup และติดตั้ง environment ของ VS code และ monogame ผ่าน terminal และ script ที่อาจารย์ให้มา และทำการ comit ตัว project ของเราและ publish ไปยัง github ของตนเอง 

## ③ ปัญหา / ข้อสงสัย — สิ่งที่ยังไม่เข้าใจหรืออยากสำรวจต่อ
ส่วนใหญ่จะเป็นเรื่องที่อาจารย์ให้ตวรจสอบซ้ำมากกว่าว่าโปรแกรมรันถูกไหม เช่น การใช้คำสั่ง git config --global user.name "ชื่อ นามสกุล"
git config --global user.email "email@example.com" git config --global --list อาจารย์ไม่ได้อธิบายให้ชัดเจนว่าคำสั่งใช้ยังไง ทำงานอะไร ใช้เพื่ออะไรเป็นต้น
